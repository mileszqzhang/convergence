简介¶

在这里我们将会使用一些数据可视化的方法来分析一下近几十年来一些有关电影的一些信息，我们使用的数据集来自“电影数据库”（TMDb，The Movie Database），当中涵盖了一万多条从上世纪60年代到2015年的电影的信息记录，记录了包括电影的受欢迎程度、制作成本、票房收入、演员、导演、电影时长、上映年份等信息。

我的思考将会从这几个关键的信息点出发：电影的受欢迎程度、电影的票房收入、以及电影的评分，基于此去尝试探讨这些特征与其他特征之间的联系。因此我提出了以下问题：

  • 哪些类型的电影最受欢迎？
  • 电影票房收入与哪些因素有关联？
  • 人气较高的电影内容上都有什么特点?
  • 哪些导演的作品评分最高？

    In [22]:
    #定义一个函数来实现该功能
def keywords_count(series):
    keyword_series=pd.Series()
    for i in series.str.split("|"):
        for j in i:
            if j in keyword_series:
                keyword_series[j]+=1
            else:
                keyword_series[j]=1
    return keyword_series

    结论¶

    最后根据我们已经做过的调查，来尝试回答最初提出的四个问题：

  • 最受欢迎的电影类型
    • 受欢迎的电影分类通常有冒险、科幻、悬念、动画、动作、家庭等。

  • 电影票房收入与哪些因素存在关联
    • 如可视化结果那样，我们可以看到电影票房收入与电影的受欢迎程度、制作成本预算、评分次数等有关，其中与评分次数有关的原因可能是因为票房收入较高的电影其人气较高，因此观看并参与评分的人数较多。另外从散点图上来看，并未发现电影票房收入与电影评分高低有密切关联。

  • 人气较高的电影的特征
    • 在人气较高的电影中，关键词“基于小说”的出现次数最多，其次依次是性、女导演、反乌托邦、纽约、监狱、音乐、体育、伦敦、独立电影、谋杀等。这可以显示出观众比较感兴趣的故事题材内容。

  • 作品好评较多的导演
    • 第三部分的可视化结果已经展示了作品所获平均评分最高的十位导演，依次是Mark Cousins、Carl Tibbetts、David Mallet、Saul Swimmer等人。

    需要注意的是，以上结果仅仅是通过可视化得出的初步结论，因此该结论只是暂时的。这些结论尚未经过统计验证，未来需要通过相关统计学方法对其进一步验证。
